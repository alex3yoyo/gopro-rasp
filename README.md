# gopro-rasp #

A simple library to control a GoPro Hero2 camera with the Raspberry Pi

## Wiring ##

- GPIO pin 23 to GoPro pin 12
- GPIO pin 24 to GoPro pin 13
- GPIO pin 25 to GoPro pin 24
